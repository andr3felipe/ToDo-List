### Como iniciar
Abra o terminal e execute:

```
git clone https://github.com/andr3felipe/ToDo-List.git
cd todo-list
npm i
npm run dev
```

#### Digite [http://localhost:3000](http://localhost:3000) no seu navegador e pronto...
